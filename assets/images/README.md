# Fondos del Juego

Este directorio contiene las imágenes de fondo del juego.

## Archivos requeridos:

### 1. `space_background.png`
- **Descripción**: Fondo espacial para el modo nave
- **Dimensiones**: 800x600 píxeles (o cualquier tamaño, se escalará automáticamente)
- **Contenido sugerido**: 
  - Espacio profundo con estrellas de distintos tamaños
  - Planetas y galaxias en el fondo
  - Soles/estrellas brillantes
  - Nebulosas y efectos espaciales

### 2. `cyberpunk_background.png`
- **Descripción**: Fondo cyberpunk futurista para el modo plataformas
- **Dimensiones**: 800x600 píxeles (o cualquier tamaño, se escalará automáticamente)
- **Contenido sugerido**:
  - Rascacielos futuristas con neones
  - Luces de neón de colores vibrantes (rosa, azul, verde, púrpura)
  - Ambiente nocturno cyberpunk
  - Edificios altos con ventanas iluminadas

## Cómo generar las imágenes con IA:

### Opción 1: DALL-E / ChatGPT
```
Prompt para espacio:
"Fondo espacial estilo videojuego 2D, espacio profundo con planetas, galaxias, estrellas de distintos tamaños, soles brillantes, nebulosas, estilo pixel art o ilustración digital, 800x600 píxeles, colores vibrantes"

Prompt para cyberpunk:
"Fondo cyberpunk futurista estilo videojuego 2D, rascacielos futuristas con luces de neón de colores vibrantes, ambiente nocturno, estilo pixel art o ilustración digital, 800x600 píxeles, neones rosa azul verde púrpura"
```

### Opción 2: Midjourney
```
/imagine prompt: space background 2D game style, deep space with planets galaxies stars of different sizes bright suns nebulas, pixel art style, 800x600, vibrant colors

/imagine prompt: cyberpunk futuristic background 2D game style, futuristic skyscrapers with neon lights vibrant colors, night atmosphere, pixel art style, 800x600, neon pink blue green purple
```

### Opción 3: Stable Diffusion / Automatic1111
Usa los mismos prompts mencionados arriba.

## Notas:
- Si las imágenes no están presentes, el juego usará los fondos por defecto (estrellas animadas)
- Las imágenes se escalarán automáticamente al tamaño de la pantalla
- Formatos soportados: PNG, JPG (PNG recomendado para transparencias)
